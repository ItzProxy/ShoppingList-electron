# ShoppingList-electron
### Simple Electron application to get used to use of JavaScript, some ES6 syntax and Electron's framework

#### To run in development
```bash
npm start
```

#### To package for release

* Windows
```bash
npm run package-win
```

* Mac
```bash
npm run package-mac
 ```

* Linux
```bash
npm run package-linux
```
